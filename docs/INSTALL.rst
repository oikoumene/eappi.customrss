eappi.customrss Installation
----------------------------

To install eappi.customrss using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``eappi.customrss`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        eappi.customrss

* Re-run buildout, e.g. with:

    $ ./bin/buildout
